# Recursive Chip Design

The emerging practice of using current AI models to design the next generation of chips, systems, and low-level software needed to run the next models — creating a recursive improvement loop. Current chip design takes 3 years from ideation to production; AI-assisted design is the only feasible way to compress this cycle and keep pace with model advancement.
