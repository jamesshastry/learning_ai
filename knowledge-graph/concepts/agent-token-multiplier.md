# Agent Token Multiplier

The phenomenon where agentic AI workloads (doing tasks vs answering questions) consume 10x+ more tokens than chat-based interactions, but deliver 100x+ more value. This drives demand for inference compute exponentially and is the primary reason token consumption curves have gone parabolic.
