# Code as AGI-Complete Domain

The thesis held by many AI researchers that coding models represent general intelligence because every task can ultimately be reduced to writing and executing code. This explains why Claude and other models increasingly write code to accomplish non-coding tasks (slide generation, data analysis) rather than using specialized tool calls.
