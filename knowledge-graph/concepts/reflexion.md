# Reflexion

A learning method where agents reflect on failed task attempts, generate textual insights about what went wrong, and persist these reflections as long-term memory to improve future performance — a new form of learning through language rather than gradient descent.

## Related Concepts

- [[Long-Term Memory]] (requires) — Reflexion insights are stored in and retrieved from long-term memory.
- [[Reinforcement Learning]] (alternative to) — Replaces scalar rewards and gradient updates with textual feedback and memory.
