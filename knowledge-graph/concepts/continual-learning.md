# Continual Learning

The emerging capability for AI models to learn from production interactions with sparse, implicit reward signals — such as whether a user accepted or reverted a code suggestion. Represented by Cursor's composer model (online training from user behavior) and Applied Compute's context basing (offline agents extracting learnings from traces). The "holy grail" and next major bottleneck in AI development.
