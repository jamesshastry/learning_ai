# CIA Triad for AI

Extension of the classic CIA security framework to agentic AI: Confidentiality (system secrets, prompts, user data, model parameters), Integrity (model parameters, data accuracy), Availability (model performance, service reliability). Also includes AI safety (preventing harm to external world).
