# Context-Driven Capability Discovery

The process of discovering new model capabilities through deployment feedback. Flux 1's open-weight release revealed that users were training LoRAs for character consistency, revealing latent demand for image editing that wasn't in the original product plan. Led to Flux Context within 60 days.
