# Memory Systems (Agentic Systems)

Multi-layered knowledge management architectures that store and retrieve information for AI agents, often combining knowledge wikis, vector search, graph databases, and an epistemology system.

## Key Points from Sources

- **cs153 Lecture 03**

## Related Concepts
