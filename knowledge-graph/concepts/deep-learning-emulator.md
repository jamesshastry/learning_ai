# Deep Learning Emulator

A neural network trained on synthetic data generated by physics simulators (solving differential equations) that produces equivalent predictions at 1000x+ speedup. Exploits the unlimited availability of simulation data while amortizing expensive data generation over millions of fast inference calls.
