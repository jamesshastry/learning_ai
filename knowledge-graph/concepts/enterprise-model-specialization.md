# Enterprise Model Specialization

The practice of taking frontier foundation models and training specialized versions optimized for specific enterprise tasks using proprietary data and enterprise-specific evaluations. General models set the floor; specialized models set the ceiling for competitive differentiation. Examples: DoorDash menu extraction, Cognition/Windsurf bug detection, Ramp spreadsheet search.
