# Voyager

The first LLM-powered open-ended embodied agent in Minecraft that continuously explores, acquires diverse skills by writing executable code, stores them in a retrievable skill library, and composes learned skills for increasingly complex tasks.

## Related Concepts

- [[Eureka]] (complementary) — Voyager explores and builds skills; Eureka trains low-level motor control.
- [[Long-Term Memory]] (instantiation of) — Voyager's skill library is a form of procedural long-term memory.
